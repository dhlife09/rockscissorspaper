# rockscissorspaper
파이썬으로 즐기는 가위바위보 게임!

---
* 변수 정보

 a = 가위
 
 b = 바위
 
 c = 보
 
 
 w = 승리 #이겼을 때
 
 l = 패배 #졌을 때
 
 d = 비겼군요 #비겼을 때
 
 
 m = 사용자의 input
 
 k = random 값
 
 i = k 값을 바탕으로 변환된 문자
 

* 게임결과의 경우

 1. 무승부
 
 - 사용자의 input == random값을 바탕으로 변환된 문자
 
 2. 승리
 
 * 사용자/랜덤/게임결과
 - 가위/보/승리
 
 - 바위/가위/승리
 
 - 보/바위/승리
 
 
 3. 패배
 
 - 위 경우에 해당 없음.
